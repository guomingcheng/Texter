# 简介

#
maven 是用来构建项目与简化 jar 包的管理

maven 是一款服务于 java 平台的自动化构建工具，对于一个大工程的项目使用 maven 来管理也是很方便的，因为 maven 可以创建多个子项目。maven 也统一了 jar 依赖。

# 安装教程

# 1、检查是否安装 jdk

maven 是使用 java 编写的自动化工具，所以 maven 运行需要 jre 环境

# 2、下载安装包

打开路径下载 : https://maven.apache.org/download.cgi

压缩包下载后，解压在一个目录即可。

# 3、配置环境变量

在 电脑右键进入/高级系统设置/用户环境变量 中设置以下内容。

MAVEN_HOME=C:\EnvironmentalScience\apache-maven-3.6.1 # 这个我解压 maven 后的路径
path=C:\EnvironmentalScience\apache-maven-3.6.1\bin # 如同

# 4、检查 maven 环境配置是否成功

命令: mvn -v
